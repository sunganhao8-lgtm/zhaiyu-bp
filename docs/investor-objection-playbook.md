# 宅域投资人异议话术库

> 对应决策：DEC-028  
> 来源：2026-07-01 与外部反馈者关于计划书落地路径的聊天记录  
> 用途：合伙人、投资人沟通、路演问答、BP 现场讲解

## 1. 聊天记录提炼

外部反馈者的核心判断是：方向不错，但投资人会追问“怎么落地”。对话中出现的关键问题：

| 对方问题 / 反馈 | 背后真实疑虑 | 宅域应对方向 |
|---|---|---|
| “你准备怎么落地？是自己做实体，还是打造平台？” | 项目是否太虚，平台化是否缺少起点 | 先做实体信任锚点，再把流程、档案和方案平台化 |
| “打造个人 IP、精品案例吸引客户，推荐平台最快” | 获客路径是否清晰 | 自媒体是放大器，实体店是信任与转化锚点 |
| “视频还不够好，不吸引目标客户” | 内容获客是否过度乐观 | 不赌泛流量爆款，做本地真实案例和可追责服务记录 |
| “为什么客户敢让你上门？” | 上门服务存在安全和信任门槛 | 用门店地址、全程录像、服务前后拍照、流程标准化降低顾虑 |
| “服务只是引流，卖货才是赚钱” | 收益模型是否成立 | 服务破冰，商品选品、套餐方案、上门改造和复购贡献利润 |
| “装修要做暖心一点” | 门店体验是否匹配信任定位 | 空间调性要让客户感到安全、干净、温暖、可停留 |

## 2. 一句话总回答

宅域不是先做空的平台，也不是只做一个宠物店；我们先用实体门店建立信任，用上门服务和内容积累真实案例，用客户发言档案找到反复出现的家庭需求，再把这些需求做成商品、套餐、上门方案和可复制 SOP。

## 3. 投资人标准问答

### Q1：你到底是做实体店，还是做平台？

**推荐回答**：  
我们先做实体店，但目标不是停留在单店。实体店是信任锚点，负责让客户放心、体验商品、完成交付和售后；线上自媒体负责获客和放大；客户发言档案负责沉淀需求；方案库和 SOP 负责复制。平台化不是第一步，而是等这些资产跑通后的结果。

**不要这样说**：  
“我们以后要做一个平台。”  
这句话太空，投资人会继续追问平台从哪里来、为什么用户要来、凭什么留存。

### Q2：为什么一定要开实体店？成本不是更重吗？

**推荐回答**：  
宠物接送洗护、上门桌搭、上门搭建都属于强信任服务。没有实体地址、没有现场体验、没有可追责主体，客户第一反应就是安全问题。门店不是单纯房租成本，而是信任锚点、体验场、内容片场、售后据点和商品转化场。

**补充例子**：  
如果一个没有实体店的人说可以上门按摩、接孩子、接宠物，大多数客户不会放心。宅域用门店把“我是谁、在哪里、出了问题找谁”先回答清楚。

### Q3：视频做得不够好，怎么吸引目标客户？

**推荐回答**：  
我们不把增长押在单条爆款视频上，而做本地真实内容：接送洗护全流程、门店实拍、宠物状态前后对比、客户反馈、电竞桌搭改造前后对比、商品实测和社区探店互推。内容目标不是泛流量，而是让附近客户知道“这家店真实、能找到、敢交付”。

**验证指标**：
- 同城有效咨询数
- 私信后到店率
- 视频带来的客户发言档案数量
- 内容触发的洗护 / 桌搭 / 商品成交

### Q4：客户为什么敢让你上门或接走宠物？

**推荐回答**：  
我们把信任拆成流程：到店登记、客户授权、临时密码或当面交接、上门前后拍照、接送全程录像、洗护全程可追溯、服务完成后回访。客户买的不是一次洗护，而是省心、省力和可追责。

**关键表达**：  
“我们不是让客户盲目信任陌生人，而是用门店、流程和证据把风险降下来。”

### Q5：服务利润低，怎么赚钱？

**推荐回答**：  
服务负责破冰和建立信任，利润来自四层：第一层是洗护、美容、寄养覆盖基础现金流；第二层是猫粮猫砂、清洁用品、桌搭配件等商品附加；第三层是电竞房、收纳、智能设备等上门方案；第四层是客户发言档案带来的复购和转介绍。

**一句话**：  
服务是入口，信任是资产，商品和方案是利润。

### Q6：开店前三个月怎么证明方向对？

**推荐回答**：  
前三个月不证明全国复制，只证明单点模型成立。我们看五个指标：有效咨询数、到店体验数、接送洗护成交数、商品附加率、复购/转介绍数。只要客户愿意咨询、愿意到店、愿意让我们上门、愿意复购和买商品，就说明实体信任锚点成立。

## 4. 路演时的 60 秒版本

宅域不是先做一个空的平台。我们先做一个社区实体样板店，因为宠物接送洗护和上门桌搭都是强信任服务，没有门店客户不会放心。门店负责信任、体验、交付和售后；线上账号负责获客和案例放大；每一次咨询、吐槽和服务反馈都会进入客户发言档案。我们从档案里找高频需求，再做商品、套餐、上门方案和 SOP。前三个月我们只验证五件事：有没有咨询、有没有到店、有没有上门成交、有没有商品附加、有没有复购转介绍。

## 5. 合伙人必须统一的表达

- 不说“我们要做很多业务”，改说“我们从客户反复说出的需求里长出新业务”。
- 不说“我们以后做平台”，改说“我们先做实体信任锚点，再把档案、SOP、方案库和商品库平台化”。
- 不说“靠短视频爆火”，改说“靠本地真实案例和可追责服务记录建立信任”。
- 不说“服务本身很赚钱”，改说“服务破冰，商品和方案贡献长期利润”。
- 不说“客户会相信我们”，改说“门店地址、流程、录像、回访和售后让客户敢第一次尝试”。
