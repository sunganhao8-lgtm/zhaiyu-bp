## 1. 项目重新定位

宅域不应被表述为“单纯宠物店”，也不应被表述为“什么都能做的万能门店”。

更准确的定位是：

**可信服务入口 + 客户发言档案 + 家庭需求产品化系统。**

- 宠物洗护、美容、寄养和用品零售承担稳定现金流，也承担第一次信任破冰。
- 电竞房样板间承担理想居住环境的可视化表达、内容传播和高客单上门搭建转化。
- 自媒体账号矩阵承担持续获客，也承担需求测试。
- 客户发言档案承担长期增长：把真实抱怨、偏好、预算和生活场景沉淀成可复用资产。
- 其他业务先进入观察池，只有从客户档案中反复出现并通过 MVP 验证后，才能进入主模型。

一句话对外表达：

**我们先用可信服务破冰，再把客户说出口的家庭麻烦事，做成能成交、能复购、能复制的方案和产品。**

## 2. 客户发言档案系统

### 2.1 为什么它是主线

服务不是终点，服务是破冰。

陌生客户不会一开始就相信“宅域能解决家里的所有事”。他们只会先因为一个明确痛点来找我们：猫洗不了、没时间接送、房间太乱、桌搭难看、灯光不舒服、东西太多不知道买哪个。

一旦我们把这次小事做好，客户就会开始说更多真实话。这些话就是宅域最值钱的原材料。

### 2.2 档案字段

| 字段 | 记录内容 | 价值 |
|---|---|---|
| 基础画像 | 住区、家庭结构、租房/自住、宠物情况、预算区间 | 判断服务半径和消费能力 |
| 原话记录 | “我最烦的是…”“我一直想…”“网上买怕踩雷…” | 保留真实需求语言，用于内容和销售话术 |
| 场景问题 | 洗护、喂养、清洁、桌搭、收纳、智能设备、维修安装 | 判断能否形成新业务 |
| 解决过程 | 本次服务做了什么、客户哪里满意/不满意 | 优化 SOP |
| 后续机会 | 可推荐商品、套餐、上门服务、复购提醒 | 形成转化和复购 |

### 2.3 运营闭环

| 环节 | 动作 | 结果 |
|---|---|---|
| 服务破冰 | 洗护、接送、桌搭诊断、样板间体验 | 第一次信任 |
| 发言入库 | 每次咨询和服务后记录 3-5 条客户原话 | 客户发言档案 |
| 需求归类 | 每周复盘高频需求 TOP10 | 内容选题 / 商品方向 |
| 方案打样 | 用低库存 SKU、上门小单、样板间测试 | 验证是否能成交 |
| 产品化 | 形成套餐、好物清单、SOP、会员权益 | 毛利、复购、转介绍 |

### 2.4 合伙人职责变化

合伙人不是只“洗猫、拍视频、装灯带”。合伙人要成为板块主理人：

- 会服务：把第一次体验做好；
- 会倾听：听懂客户真正没说完的麻烦；
- 会记录：把原话和场景沉淀下来；
- 会复盘：判断哪些需求有重复性；
- 会产品化：把需求做成套餐、商品、内容和 SOP。

## 3. 首期业务分层

| 层级 | 业务 | 当前处理 |
|---|---|---|
| 首期必做 | 宠物洗护、美容、基础寄养、宠物用品零售 | 写入开业 SOP 和日收入模型 |
| 样板展示 | 电竞房样板间、上门桌搭/电竞房搭建、低库存桌搭好物 | 写入内容和转化模型，不按散客收费模型计算 |
| 验证后做 | 上门喂养、代理送医、宠物社群活动 | 先做小样本测试，明确责任和赔付边界 |
| 观察池 | 减脂餐、咖啡/果茶、AI 培训、情侣/单身活动 | 不计入首期主模型，单独做 MVP |
| 暂不做或需资质 | 疫苗注射、宠物诊疗、酒馆、食品餐饮、学生自习收费、游戏厅式经营 | 资质核查前不执行 |

首期开业时只承诺第一层和第二层，避免业务发散。

## 4. 电竞房样板间设计

### 3.1 定位

电竞房区域不是“给客人来打游戏的房间”，而是：

- 合伙人日常工作/娱乐区；
- 小红书、抖音、视频号的内容拍摄点；
- 桌搭、投影、大屏、氛围灯、收纳、手办展示样板间；
- 客户到店看效果后的上门搭建转化入口；
- 宠物店等待区的差异化记忆点。

### 3.2 面积和配置

建议控制在 **5-8㎡**，优先做“视觉冲击强但低成本”的样板角：

- 一面展示墙：灯带、洞洞板、理线槽、支架、手办/模型陈列。
- 一个桌搭工位：显示器/键鼠/支架/椅子/灯光。
- 一个大屏或投影：用于样板展示、观影、内容拍摄背景。
- 一个小型商品陈列区：只放高频、低库存、易复购或可代采的 SKU。

不建议首期投入大沙发、大展示柜、多工位重资产配置。

### 3.3 产品化服务

| 产品 | 建议价 | 说明 |
|---|---:|---|
| 桌搭诊断/方案 | 99-199 | 可抵扣后续服务，适合从内容咨询转化 |
| 基础理线改造 | 399-699 | 插排、理线槽、支架、灯光、桌面清理 |
| 大屏/投影方案 | 999-2999 | 设备代采 + 安装调试 + 使用教学 |
| 全屋电竞房搭建 | 2999+ | 需要单独报价，不承诺标准价 |
| 桌搭好物带货 | 按毛利 | 灯带、支架、插排、理线、氛围灯等 |

表格中“上门搭电竞房 200 元/次”的口径偏低，应改为“入门服务或诊断费”，不能代表完整业务收入。

## 5. 财务模型修正

当前 Excel 的关键结果：

- 月固定成本：`9850 元`
- 日固定成本：`328.33 元`
- 日目标收入：`750 元`
- 当前计划日收入：`699.5 元`
- 缺口：`50.5 元/天`

当前模型的问题：

1. “上门搭电竞房”只贡献 `12.5 元/天`，无法证明电竞区的面积价值。
2. 宠物基础业务约 `322 元/天`，加驱虫、疫苗、送医后约 `402 元/天`，但其中存在合规风险项。
3. 减脂餐、饮品、AI 培训等把日收入拉高，但与门店主业关系弱。
4. 股权表中 `12 万`、`13.5 万` 两个口径并存，需统一。

建议把财务模型拆成三张表：

| 表 | 用途 |
|---|---|
| 首期现金流表 | 只放宠物洗护、美容、寄养、用品零售 |
| 电竞转化漏斗表 | 曝光、咨询、到店看样板、成交率、客单价、毛利 |
| 观察池 MVP 表 | 减脂餐、饮品、AI 培训、活动策划等逐项验证 |

电竞样板间的核心指标不应是“每天几个人来玩”，而应是：

- 每周发布内容数；
- 每周咨询数；
- 到店看样板人数；
- 上门方案成交数；
- 单均毛利；
- 周边带货 GMV 和毛利。

## 6. 股权、抽成和分红机制

当前机制容易混淆三件事：

- 股东按股权分红；
- 人员按具体服务拿抽成；
- 固定岗位是否有工资/补贴。

建议规则：

1. **业务抽成**：只对明确 owner 的服务项目生效，比如洗护、上门桌搭、上门喂养。
2. **岗位补贴**：如果某人长期承担店内接待、清洁、社群、记账，应按岗位补贴或固定责任计入，不要混在股权里。
3. **股东分红**：不日结，建议月度结账、季度分红。
4. **安全资金**：账户保留至少 3 个月固定成本后再分红，约 `9850 × 3 = 29550 元`。
5. **人力股**：店长D 5% 应明确是分红权、表决权、退出条件、未履职处理方式。

建议补一份《合伙协议核心条款》，至少包括：

- 出资额和股权；
- 分红节奏；
- 业务抽成规则；
- 财务公开频率；
- 大额支出审批；
- 股权转让；
- 退出机制；
- 业务内私活边界；
- 关系变化不影响公司机制的处理方式。

## 7. 自媒体账号矩阵

不要把所有内容混在一个账号里。建议三条线。每条内容都来自客户发言档案，而不是凭空想选题：

| 账号线 | 主题 | 目标 |
|---|---|---|
| 宠物洗护号 | 洗猫洗狗、护理知识、门店卫生、宠物故事 | 到店洗护、寄养、私域群 |
| 电竞房样板号 | 桌搭改造、投影大屏、理线、氛围灯、好物推荐 | 上门搭建、周边带货 |
| 创业复盘号 | 低成本开店、合伙机制、AI 做 BP、财务模型 | 合伙人、潜在客户、项目背书 |

新增一条内部要求：每周从客户发言档案里选出 10 条原话，至少转化成 3 条内容选题、1 个小商品测试、1 个服务套餐优化。

每条内容都要有明确转化动作：

- 宠物内容：加微信预约洗护/进养宠群；
- 电竞内容：发房间照片领取桌搭方案；
- 创业内容：领取项目 BP/咨询空间搭建。

## 8. 开业前 30 天执行清单

### 第 1 周：租约和合规

- 明确房租、押金、免租期、递增条款。
- 核查营业执照经营范围。
- 明确宠物洗护、寄养、用品零售可执行范围。
- 暂停疫苗注射、诊疗、食品餐饮、酒类、游戏厅式经营。

### 第 2 周：空间和采购

- 出 49㎡ 平面图，明确湿区、干区、电竞样板区。
- 宠物设备只买开业必需品。
- 电竞样板间先做一个工位、一面墙、一套大屏/投影展示。
- 建立设备台账和资产归属。

### 第 3 周：价格和 SOP

- 完成洗护、美容、寄养价目表。
- 完成上门桌搭服务套餐。
- 写清洗护接待、消毒、寄养、售后、投诉处理流程。
- 明确客户拍摄授权和视频素材使用规则。

### 第 4 周：内容和试营业

- 宠物号连续发布 10 条本地内容。
- 电竞号连续发布 10 条桌搭内容。
- 开业活动只做免费剪指甲、半价洗护、桌搭诊断名额。
- 每天记录进店、咨询、成交、复购、投诉。
- 每天至少记录 5 条客户原话，建立第一版客户发言档案。
- 每周复盘一次“客户说得最多的 10 个麻烦”。

## 9. 当前最优改法

1. 不推翻“宠物 + 电竞”的方向。
2. 把电竞区从“娱乐区”改成“样板间和成交入口”。
3. 把业务从“全部都做”改成“分层验证”。
4. 把表格从“日收入凑 750”改成“宠物现金流表 + 电竞漏斗表”。
5. 把股权机制从“抽成和分红混算”改成“抽成、岗位、分红分离”。
6. 把品牌主线从“我们能做很多事”改成“我们通过服务听懂客户，再把客户需求产品化”。

这套改法能保留项目的想象力，也能让合伙人清楚知道：第一阶段到底做什么、怎么赚钱、什么暂时不能碰。
